# Social Chat

Social Chat ek all-in-one social media aur messaging app hai jo WhatsApp ki messaging, Instagram ka photo feed aur stories, aur TikTok ke swiping reels ka behtareen mix pradan karta hai. 

## Features
- **Home Feed**: Apne doston ki stories dekhein aur naye posts like/comment karein (Instagram style).
- **Chats**: WhatsApp ki tarah apne doston se real-time chat karein.
- **Reels**: TikTok style mein vertical video swipe karein aur trendings videos enjoy karein.
- **Profile**: Apna profile manage karein, posts aur followers track karein.

## Tech Stack
- Flutter
- Material 3 Design

## Setup
App ko run karne ke liye:
```bash
flutter pub get
flutter run
```
